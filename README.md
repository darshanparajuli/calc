# calc

A simple command line calculator written in Rust.

#### Example
```
$ calc
λ 1+2
=> 3

λ a = 23
=> a = 23

λ a*2
=> 46

λ b = sin(a) * 20^2
=> b = -338.4881616700683

λ b
=> -338.4881616700683

λ ans
=> -338.4881616700683

λ ans / 100
=> -3.384881616700683

λ log10(10)
=> 1

λ log2(8)
=> 3

λ log10(100)
=> 2

λ ln(e)
=> 1

λ sin(pi/2)
=> 1

λ exit
```
